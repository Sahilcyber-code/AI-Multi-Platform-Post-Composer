# 🚀 AI Multi-Platform Post Composer

An AI-powered full-stack web application that allows users to create, manage, preview, and publish social media posts across multiple platforms from a single dashboard.

Built using the MERN stack with secure authentication, MongoDB Atlas, and an intuitive modern UI.

---

## 📸 Features

### 🔐 Authentication
- User Registration
- User Login
- JWT Authentication
- Password Encryption
- Protected Routes

### ✍️ AI Post Composer
- Create posts for multiple social media platforms
- Platform selection
- Character counter
- Platform-specific validation
- AI writing assistance
- Responsive text editor

### 📱 Live Preview
- Twitter/X Preview
- LinkedIn Preview
- Facebook Preview
- Instagram Preview

### 📂 Media Upload
- Upload images
- Preview uploaded media
- Store uploaded files

### 📝 Draft Management
- Save Drafts
- Edit Drafts
- Delete Drafts

### 📊 Dashboard
- User Profile
- Post History
- Analytics Section
- Recent Activity

### 🎨 Modern UI
- Responsive Design
- Clean Dashboard
- Smooth User Experience

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/ai-multi-platform-post-composer.git
```

Move into the project

```bash
cd ai-multi-platform-post-composer
```

---

## Install Frontend

```bash
npm install
```

---

## Install Backend

```bash
cd backend
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the **backend** folder.

Example:

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

---

# ▶️ Running the Project

### Start Backend

```bash
cd backend
npm run dev
```

### Start Frontend

```bash
npm run dev
```
